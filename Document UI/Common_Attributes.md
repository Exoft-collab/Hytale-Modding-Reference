# Common Attributes

Properties that apply to most UI components in the project.

## 1. Anchor (Dimensions & Alignment)

| Key | Description | Example |
| :--- | :--- | :--- |
| `Width` | Pixel width. | `Width: 100` |
| `Height` | Pixel height. | `Height: 50` |
| `Full` | Stretch to fit parent (0 or 1). | `Full: 0` |
| `Horizontal` | Sets both Left/Right. | `Horizontal: 1` |
| `Vertical` | Sets both Top/Bottom. | `Vertical: 0` |
| `Top` | Distance from top. | `Top: 10` |
| `Bottom` | Distance from bottom. | `Bottom: 5` |
| `Left` | Distance from left. | `Left: 20` |
| `Right` | Distance from right. | `Right: 0` |

## 2. Padding (Inner Spacing)

| Key | Description | Example |
| :--- | :--- | :--- |
| `Full` | Uniform padding. | `Full: 10` |
| `Horizontal` | Left/Right padding. | `Horizontal: 20` |
| `Vertical` | Top/Bottom padding. | `Vertical: 10` |
| `Left` / `Right`| Specific horiz spacing. | `Left: 5` |
| `Top` / `Bottom` | Specific vert spacing. | `Top: 8` |

## 3. Colors

- Hex: `#FFFFFF`
- Alpha (Decimal): `#ffffff(0.85)`
- Block: `(Color: #FFFFFF)`

## 4. Other Confirmed Keys

- `Visible`: Boolean (Show/Hide).
- `FlexWeight`: Number (Dynamic sizing).
- `TooltipText`: String (Hover text).
- `TextTooltipShowDelay`: Number (Delay for tooltip).
- `ContentHeight`: Number (Custom height adjust).

# ItemIcon

Used for displaying Minecraft/Hytale item icons.

## Confirmed Properties

- `Visible`: Boolean - Toggle visibility.
- `Anchor`: Block - Dimensions and positioning.

## Comprehensive Example

```hytale
ItemIcon #SlotIcon {
  Anchor: (Width: 64, Height: 64, Top: 4, Left: 4);
  Visible: true;
}
```
